## 2. Постановка каждой задачи  

### Задача 1: Сумма двух чисел (task01.c)  
**Цель:** отработать ввод/вывод и переменные типа int.  
**Вход:** два целых числа a и b.  
**Выход:** их сумма.  
**Ограничения:** -10000 <= a, b <= 10000.  
**Пример:**  
Ввод: `7 5`  
Вывод: `12`

---

### Задача 2: Простая формула (task02.c)  
**Цель:** вычислить математическое выражение с double.  
**Формула:** u = (x + y) / 2 + x * y  
**Вход:** два вещественных числа x, y.  
**Выход:** значение u с точностью 3 знака после запятой.  
**Ограничения:** -100.0 <= x, y <= 100.0.  
**Пример:**  
Ввод: `2.0 4.0`  
Вывод: `11.000`

---

### Задача 3: Проверка чётности (task03.c)  
**Цель:** использовать условный оператор if.  
**Вход:** целое число n.  
**Выход:** `EVEN`, если число чётное, иначе `ODD`.  
**Ограничения:** |n| <= 1000000.  
**Пример:**  
Ввод: `9`  
Вывод: `ODD`

---

### Задача 4: Максимум из трёх (task04.c)  
**Цель:** сравнение нескольких значений через if.  
**Вход:** три целых числа a, b, c.  
**Выход:** максимальное число.  
**Ограничения:** |a|, |b|, |c| <= 1000000.  
**Пример:**  
Ввод: `3 10 7`  
Вывод: `10`

---

### Задача 5: Сумма от 1 до n (task05.c)  
**Цель:** цикл for.  
**Вход:** целое число n.  
**Выход:** сумма 1 + 2 + ... + n.  
**Ограничения:** 1 <= n <= 10000.  
**Пример:**  
Ввод: `5`  
Вывод: `15`

---

### Задача 6: Квадраты элементов массива (task06.c)  
**Цель:** работа с одномерным статическим массивом.  
**Вход:** число n, затем n целых чисел.  
**Выход:** n чисел — квадраты элементов в том же порядке.  
**Ограничения:** 1 <= n <= 100, |a[i]| <= 1000.  
**Пример:**  
Ввод: `4` и `1 -2 3 4`  
Вывод: `1 4 9 16`

---

### Задача 7: Разворот массива (task07.c)  
**Цель:** индексация массива и обмен элементов.  
**Вход:** число n, затем n целых чисел.  
**Выход:** элементы в обратном порядке.  
**Ограничения:** 1 <= n <= 100, |a[i]| <= 1000.  
**Пример:**  
Ввод: `5` и `10 20 30 40 50`  
Вывод: `50 40 30 20 10`

## 5. Код программы  

### task01.c
```c
#include <stdio.h>

int main() {
    int a, b, sum;
    scanf("%d %d", &a, &b);
    sum = a + b;
    printf("%d\n", sum);
    return 0;
}
```
### task02.c
```C
#include <stdio.h>

int main() {
    double x, y, u;
    scanf("%lf %lf", &x, &y);
    u = (x + y) / 2 + x * y;
    printf("%.3lf\n", u);
    return 0;
}
```
### task03.c
```C
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);
    if (n % 2 == 0)
        printf("EVEN\n");
    else
        printf("ODD\n");
    return 0;
}
```
### task04.c
```C
#include <stdio.h>

int main() {
    int a, b, c, max;
    scanf("%d %d %d", &a, &b, &c);
    max = a;
    if (b > max) max = b;
    if (c > max) max = c;
    printf("%d\n", max);
    return 0;
}
```
### task05.c
```C
#include <stdio.h>

int main() {
    int n, sum = 0;
    scanf("%d", &n);
    for (int i = 1; i <= n; i++) {
        sum += i;
    }
    printf("%d\n", sum);
    return 0;
}
```
### task06.c
```C
#include <stdio.h>

int main() {
    int n, a[100];
    scanf("%d", &n);
    for (int i = 0; i < n; i++) {
        scanf("%d", &a[i]);
    }
    for (int i = 0; i < n; i++) {
        printf("%d ", a[i] * a[i]);
    }
    printf("\n");
    return 0;
}
```
### task07.c
```C
#include <stdio.h>

int main() {
    int n, a[100], temp;
    scanf("%d", &n);
    for (int i = 0; i < n; i++) {
        scanf("%d", &a[i]);
    }
    for (int i = 0; i < n / 2; i++) {
        temp = a[i];
        a[i] = a[n - 1 - i];
        a[n - 1 - i] = temp;
    }
    for (int i = 0; i < n; i++) {
        printf("%d ", a[i]);
    }
    printf("\n");
    return 0;
}
```
